Fin-ray Gripper

Contact-Rich Manipulation & Bio-Inspired Design

Project Overview

−

A novel robotic manipulation system combining adaptive 3D-printed TPU fin-ray structures with an integrated 8x4 tactile sensor matrix. The system adapts passively to object geometries while providing real-time feedback for grip adjustment based on slip detection.

Hardware Architecture

+
ControllerParallax Propeller
MultiplexerCD74HC4067 (16-Ch)
Sensor GridCopper Tape + Velostat
ActuatorHigh-Torque Servo

Software Stack & State Machine

+

Distributed processing across Propeller Cogs for sensor acquisition (Cog 0) and servo control (Cog 1), with Python/OpenCV for state classification.
